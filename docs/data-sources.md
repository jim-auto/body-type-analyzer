# Data Sources

## 日本人平均統計

- 女性身長平均 / 男性身長平均 / 女性バスト平均は `AGENT_TASK.md` の指定値を実装値として採用。
- 偏差値計算式は `偏差値 = 50 + 10 * (値 - 平均) / 標準偏差`。
